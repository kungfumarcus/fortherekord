# ps1/

Original PowerShell scripts that this Python application is rewriting.

## Files

- `Update-RekordboxLibrary.ps1` - Original PowerShell script for Rekordbox library processing
- `Update-Spotify.ps1` - Original PowerShell script for Spotify integration and synchronization
- Configuration files and supporting scripts

## Purpose

These scripts serve as the reference implementation and requirements source for the Python rewrite. The Python application aims to maintain identical functionality while improving maintainability and adding modern libraries.

## Key Features Analyzed

- XML parsing using PowerShell's native XML handling
- Hashtable-based data structures
- Direct XML attribute access
- Simple text processing and normalization
- Spotify API integration patterns
- Track matching algorithms
- Playlist synchronization logic

## Usage as Reference

When implementing new features in the Python version, refer to these scripts to understand:
- Expected behavior and edge cases
- Data flow patterns
- Configuration requirements
- Error handling approaches
